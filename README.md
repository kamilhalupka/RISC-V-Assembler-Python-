# RISC-V-Assembler-Python-

RISC-V Assembler using 100% Python Code.

Supports all standard RISC-V 32 Instructions (no psuedoinstructions or mult/div extensions).

Supported file types: .s, .asm

Input file is parsed line by line and converted into hex.
Output is written into a .hex file in the working directory of the application.
